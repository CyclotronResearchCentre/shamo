API reference¶

The public API of shamo only provides a subset of the classes and methods defined in the package. Here is a quick description of the exposed classes.

Finite element model¶

  • FEModel: The model itself which holds all the data and provides all the methods.

  • MeshConfig: The parameters passed to the meshing algorithm.

Problems¶

EEG¶

  • EEGForwardProblem: A problem used to generate a single leadfield matrix.

  • EEGParametricForwardProblem: A problem used to generate a parametric leadfield matrix.

  • EEGSimulationProblem: A problem to simulate sources in the brain for EEG.

Solutions¶

EEG¶

  • EEGForwardSolution: A single leadfield matrix.

  • EEGParametricForwardSolution: A parametric leadfield matrix.

  • EEGSimulationSolution: A simulation of sources in the brain for EEG.

Sources¶

  • FESource: A source defined in a finite element model.

  • EEGSource: A source defined in an EEG problem.

Distributions¶

  • ConstantDistribution: For a property with a fixed value.

  • TruncatedNormalDistribution: For a property with a bounded normal distribution.

  • UniformDistribution: For a property with uniformly distributed values.
