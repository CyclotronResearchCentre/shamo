The path to the parent directory of the solution.

n_evalsint

The number of evaluation points.

methodstr, optional

The method to solve the sub-problems. The accepted values are 'sequential', 'multiprocessing', 'job'. (The default is 'sequential')

n_procint, optional

The number of processes to solve the problem when method is set to 'multiprocessing'. (The default is 1)

skipint, optional

The number of points to skip at the beginning of the sequence. (The default is 0)
