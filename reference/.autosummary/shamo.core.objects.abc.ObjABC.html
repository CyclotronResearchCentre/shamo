shamo.core.objects.abc.ObjABC¶

class shamo.core.objects.abc.ObjABC(name, parent_path)[source]¶

Bases: dict, abc.ABC

A base class for any savable/loadable object.

All the main object can be saved to/loaded from human readable JSON files to make it easily compatible with BIDS standars and simplify interactions with other scripts.

popitem(/)¶

Remove and return a (key, value) pair as a 2-tuple.

Pairs are returned in LIFO (last-in, first-out) order. Raises KeyError if the dict is empty.

save(exist_ok=True)[source]¶

Parameters
exist_okbool, optional

If set to True, any already existing object is overriden. Otherwise, if the object already exists, a FileExistsError. The default is True.

Raises

TypeError

Return the value for key if key is in the dictionary, else default.

update([E, ]**F) → None. Update D from dict/iterable E and F.¶

If E is present and has a .keys() method, then does: for k in E: D[k] = E[k] If E is present and lacks a .keys() method, then does: for k, v in E: D[k] = v In either case, this is followed by: for k in F: D[k] = F[k]
